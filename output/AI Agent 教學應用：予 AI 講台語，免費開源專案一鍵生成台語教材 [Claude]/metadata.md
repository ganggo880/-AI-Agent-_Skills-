## 2. 影片描述（≤300 字）

台語數位教材有多難做？做過的老師都知道。這次我做了一個很難、花很久的免費開源專案，要幫本土語台語老師解決這個問題：讓 AI Agent 直接生成你需要的所有台語教材。

做法很簡單：把這個 GitHub repo 的網址丟給你的 AI Agent（我用 Claude Code 示範，Codex、AntiGravity、OpenCode 也都可以），叫它照著安裝，它就會全自動裝好所有程式，完全免費、不用申請任何 API Key。

裝好之後，你在這個專案裡跟 AI 對話，就能生成：
▸ 講義與考卷（Word，附台羅拼音、課綱指標對照）
▸ 台語語音（意傳官方發音）
▸ 台語教學影片
▸ 網頁互動遊戲（拖拉配對、聽音配對、翻牌、句型練習）
▸ 上課用的 PowerPoint 簡報

核心概念：用「一個專案資料夾」承接整套複雜工作流，你也能複製這個思維，打造專屬於你的複雜工作流專案。
