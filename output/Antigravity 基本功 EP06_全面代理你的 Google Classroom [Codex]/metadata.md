## 2. 影片內容描述（YouTube 描述欄）

還在一個一個用滑鼠點開 Google Classroom 派作業、收作業嗎？
這一集帶你把整個 Classroom 交給 AI Agent，用一句自然語言就完成全班作業派發。

本集是「AntiGravity 基本功」系列第六集。我會實際示範如何用個人帳號的 AI Agent，安全地連接你教育帳號的 Google Classroom，從零完成設定到真正派出一份作業。

影片三大重點：
1. **環境串接**：用影片說明欄的公開 repo（Classroom Agent Kit），帶著 AI Agent 完成 Google Cloud 專案、Classroom API、OAuth 同意畫面與憑證（JSON）設定。
2. **全面代理 Classroom**：用自然語言請 AI Agent 自動建立課程、派發作業、附加 repo 與 YouTube 播放清單，連收作業、批改都能交給它。
3. **網頁工具串名單**：完成網頁認證後，任何你自己做的網頁工具都能一鍵拉進 Classroom 的學生名單，不必再重新輸入。

設定雖然有點複雜，但只要做一次，往後備課與作業管理都會輕鬆非常多。
