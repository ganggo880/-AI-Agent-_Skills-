## 2. 影片描述（≤300 字，貼 YouTube 用）

生成式 AI 只會「動嘴」回你文字，但 AI Agent 是有手有腳的——它能直接在你的電腦上把事情做完。

這一集用大量實演，帶你看懂 Agent 的四隻手腳到底能做什麼：

▸ 讀檔：用「複製路徑」（Ctrl+Shift+C）精準餵檔給 AI，省 Token 又不怕找錯；一句話幫你把凌亂的下載資料夾整理乾淨。
▸ 寫程式＋出成品：Agent 自己寫 Python、自動裝套件，產出 Word 考卷、Excel 成績、PowerPoint 簡報、PDF，你完全不用碰程式。
▸ 上網：四大 Agent 上網方式比較（Codex 操控瀏覽器、Claude in Chrome、Playwright），實演 AI 上網抓會考歷屆試題，自動截圖排成考卷＋解答簡報。

還加碼分享：老師專用的 Python 工具清單（GitHub 一鍵安裝）、Codex 內建語音輸入，以及研習免費備案 Big Pickle。

最後別忘了 AI 素養：Agent 取代的是勞務，最終產出一定要經過你的專業把關。
