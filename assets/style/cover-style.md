# YouTube 封面風格指南（港居不動產開發 — 極簡現代商務風）

> 此檔是頻道封面的視覺基準。所有 `cover-image` 相關的生圖 prompt 都應依此風格撰寫，以維持頻道的高質感商務識別。

## 整體調性
**「極簡現代商務 × 專業教育」風格**：乾淨、精緻、排版大氣、富有空間感。
類似現代高端建案推廣、質感商務雜誌封面或高階線上課程（如 MasterClass）的極簡美學。

---

## 五大固定元素

### 1. 背景（Background）
- **主色**：極簡灰 (#F2F2F2)、暖白 (#FDFBF7)、淺米色 (#EAE5DB)
- **材質/特效**：極細微的磨砂水泥牆紋理、乾淨的微光漸層、柔和的陰影投射，營造出具空間感的背景
- **氣氛**：明亮、舒適、具專業度與親切感
- **嚴禁**：深藍底、放射狀霓虹光線、電子粒子、太雜亂的幾何圖形

### 2. 主標題文字（Title Typography）
- **字體**：超粗黑體（如思源黑體 Heavy、Noto Sans TC Black）或高質感明體（如源樣明體）
- **主色**：**深炭灰色** (#2C2C2C) 或**深大地色** (#4A3B32)
- **副色 (點綴)**：**質感霧金/古銅色** (#C5A880) 作為關鍵字高亮
- **描邊**：無描邊，改用適當的字距與行距，以及高對比的明亮背景來確保易讀性
- **位置**：左半邊，字體靠左對齊，佔畫面寬度約 45-50%
- **排版**：2 行，高度不超過畫面的 1/3，周圍保留充足的「留白 (White Space)」以顯大氣

### 3. 副標題 / 標籤
- **副標題**：
  - 通常位於主標下方或上方，字級較小，字體較細
  - 配色為中灰色 (#666666) 或大地色
  - 描述影片核心痛點（例：「包租代管實務操作」、「AI 房產工具應用」）
- **分類標籤**：
  - 可加上細實線的外框，或使用淡淡的米色/金色色塊反白，放置在最上方，用於系列集數標示

### 4. 人物（Persona）— 港居不動產負責人
- **位置**：**畫面右側**（佔畫面寬 35–45%）
- **構圖**：半身或 3/4 身，臉部清晰、表情專業且友善
- **特徵**：延續輸入的形象基準照中的五官、髮型與衣著神態，不可由 AI 隨意改動
- **打光**：邊緣補上「柔和暖白光」或「夕陽柔焦黃光 (soft rim light)」，使人物能自然地從極簡背景中立體浮現
- **嚴禁**：浮誇發光、霓虹補光，或將人物過度飽和化

### 5. 主題視覺（Topic Visual）— 左下側或標題旁
- 與影片主題高度相關的精緻 3D 圖示、簡約的科技/房產 logo、或是高質感的流程圖示
- **特效**：搭配柔和的落影 (drop shadow)，使其像是漂浮在極簡空間中
- **配色**：以深灰色、金銅色、米色為主，保持視覺的和諧與克制

## 主題配色規則

根據影片的主角與主題，調整背景及文字高亮色，以達到一致但有層次的視覺：

| 影片主題類別 | 背景主色 | 文字高亮色 | 主視覺發光/陰影色 |
|-------------|--------|-----------|------------------|
| **AI 工具 / 科技代管** | 明亮水泥灰 (#ECECEC) | 科技灰藍 (#3E5C76) | 柔和冷灰陰影 |
| **包租代管實務 / 租賃政策** | 溫暖米色 (#F4EFEB) | 溫潤古銅金 (#C5A880) | 柔和暖褐陰影 |
| **不動產開發 / 商業談判** | 雅致碳灰白 (#F6F5F2) | 深炭黑 (#1A1A1A) | 淺金棕陰影 |

## 構圖網格（Layout Grid）

```
┌─────────────────────────────┬─────────────────┐
│ [分類標籤]                  │                 │
│                             │                 │
│  ┃質感主標題┃               │   港居負責人    │
│  ┃第二行高亮┃               │   人物         │
│                             │   (右側)       │
│ [細字副標：影片核心要旨]     │                 │
│                             │                 │
│  [精緻 3D 主題視覺]         │                 │
└─────────────────────────────┴─────────────────┘
   ←——— 左 60% ———→            ←— 右 40% —→
```

**留白守則**：文字與人物之間、文字與邊框之間，必須保留至少 10% 的空白邊距，以展現極簡主義的呼吸感。

## Prompt 撰寫範本

生圖時請直接以此範本填空撰寫：

```
YouTube 封面 16:9，港居不動產開發的高質感極簡現代商務風格。

背景：乾淨明亮的「{BACKGROUND_COLOR}」底色，帶有極細微的現代水泥/磨砂牆面紋理，帶有柔和的自然採光，整體呈現大氣、簡約的空間感。
※「{BACKGROUND_COLOR}」依主題決定：
   - 科技代管 → 淺水泥灰 (#ECECEC)
   - 包租實務 → 溫暖米色 (#F4EFEB)
   - 商業談判 → 雅致碳灰白 (#F6F5F2)

人物：畫面右側放置這位男性負責人，姿勢為「{POSE}」，邊緣帶有柔和暖白的邊緣自然光 (soft rim light)，臉部表情「{EXPRESSION}」。
※ 人物樣貌、五官、髮型、衣著全部延續輸入的人物基準照，保持專業且親切的商務形象。

主視覺（左下角）：{TOPIC_VISUAL}，帶有乾淨的落影 (drop shadow)，與極簡背景融合。

文字排版（位於左半邊，大氣留白）：
- 頂部分類標籤：「{SERIES_LABEL}」深灰色細字，帶有精細的細實線框。
- 主標題：「{MAIN_TITLE}」深炭灰色 (#2C2C2C) 的超粗黑體中文字，其中關鍵字「{HIGHLIGHT_WORD}」使用質感霧金/古銅色 (#C5A880) 高亮，文字無描邊，2 行排列，大氣且高對比。
- 副標題：「{SUBTITLE}」深灰色細字。

整體視覺：極簡主義、大片留白、雜誌封面美學、高質感、專業商務。
嚴禁：深海軍藍底、浮誇發光、霓虹漸層、3D 描邊字、太過飽和色彩。
```
